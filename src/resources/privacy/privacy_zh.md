# 隐私政策
*最后更新时间：2026年6月27日*

本隐私政策详细说明了 Codeoba 本地桌面应用程序（“应用”）和 codeoba.com 网站（“网站”）的隐私做法。该应用和网站均由 What AI Can Do, LLC（“公司”）开发、发布和运营。

请仔细阅读本政策。如果您不同意本政策中概述的条款，请不要使用该应用或网站。

---

## 1. 核心原则：默认本地优先
> Codeoba 旨在成为一个本地优先的应用程序。所有对话记录、数据库索引、缓存和语义模型完全存储在您的本地计算机上。

这种本地优先实现的关键方面包括：
- **零远程日志:** 从本地助手目录（Claude Code, Antigravity, Cursor, OpenAI Codex, Copilot 等）汇总的会话记录在离线状态下处理，并直接在您设备上的桌面客户端运行时间内进行解析。
- **本地 SQLite 数据库与缓存:** 所有已索引的会话、日志、搜索关键字和性能数据都保留在本地缓存目录中。
- **本地向量索引:** 为了进行语义查询匹配，该应用程序在本地下载了一个量化后的 Transformer 模型（all-MiniLM-L6-v2）。所有概念计算和相似性搜索都在您本地计算机的 CPU 上运行，不向第三方 API 发送数据。

## 2. 诊断与自动更新检查
为了保持平台的正常运行和安全性，Codeoba 会执行基本的诊断更新：
- **软件自动更新检查:** 如果您明确同意，本应用会向我们的更新服务器查询最新版本。这会发送标准的网络参数（如应用程序版本和语言偏好，以及您的操作系统版本和 CPU 架构）以获取版本配置清单，但不会收集任何个人或对话数据。
- **远程遥测与诊断日志:** 为了监控服务健康状况并防止滥用 API 速率限制，更新请求会被记录到 GCP Cloud Logging 中，该日志服务会固有地识别请求来源的客户端 IP 地址。这些日志会记录操作系统详情和匿名安装 GUID。所有此类日志都会保留 30 天，然后自动被永久删除。

## 3. 网站分析与 Cookie 同意
为了分析访问者流量和监控下载率，本网站使用 Google Analytics (GA4)。默认情况下，分析跟踪完全禁用，且不会加载任何 Cookie。

当您访问本网站时，会显示一个隐私同意横幅：
- **同意接受:** 如果您接受跟踪，我们会动态加载 Google Analytics。Google Analytics 将放置持久性标识符（Cookie，如 _ga 和 _ga_<container-id>）来跟踪匿名页面浏览和下载点击。您的 IP 地址会被匿名化。
- **拒绝跟踪:** 如果您拒绝，Google Analytics 脚本将永远不会加载。不会存储任何 Cookie，也不会发送设备指纹信息。

您可以随时撤回或重置您的选择：[管理隐私偏好](#)（这将清除您保存的选择，并立即再次显示同意横幅）。

## 4. 共享您的信息
我们不会出售、出租或交易您的个人数据。我们仅与以下受信任的第三方服务提供商（子处理商）共享运行我们的更新和诊断服务所需的有限技术信息：
- **云基础设施:** 我们使用 Google Cloud Platform (GCP) Cloud Logging 存储标准的遥测日志（例如用于更新检查的 IP 地址和设备信息），以用于诊断和安全目的。这些日志会在 30 天后自动清除。

## 5. 本地数据的安全性
由于您的数据存储在本地，因此已索引对话的安全性完全取决于您自己设备的安全性。我们建议您使用标准加密工具（例如 macOS 上的 FileVault）和强密码来保护您的设备。
