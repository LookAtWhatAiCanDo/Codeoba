# 隱私權政策
*最後更新時間：2026年6月27日*

本隱私權政策詳細說明了 Codeoba 本地桌面應用程式（“應用”）和 codeoba.com 網站（“網站”）的隱私做法。該應用和網站均由 What AI Can Do, LLC（“公司”）開發、發布和營運。

請仔細閱讀本政策。如果您不同意本政策中概述的條款，請不要使用該應用或網站。

---

## 1. 核心原則：預設本地優先
> Codeoba 旨在成為一個本地優先的應用程式。所有對話記錄、資料庫索引和快取完全存儲在您的本地電腦上。

這種本地優先實現的關鍵方面包括：
- **零遠端日誌:** 從本地助手目錄（Claude Code, Google Antigravity, Cursor, OpenAI Codex, Copilot 等）彙總的對話記錄在離線狀態下處理，並直接在您裝置上的桌面用戶端運行時間內進行解析。
- **本地 SQLite 資料庫與快取:** 所有已索引的會話、日誌、搜尋關鍵字和效能數據都保留在本地快取目錄中。
- **本地預解析快取:** 所有已索引的對話均在本地解析到您使用者主目錄中的快速工作階段快取中，以便在無需外部網路呼叫的情況下立即啟動應用程式。

## 2. 診斷與自動更新檢查
為了保持平台的正常運行和安全性，Codeoba 會執行基本的診斷更新：
- **軟體自動更新檢查:** 若您明確同意，本應用程式會向我們的更新伺服器查詢最新版本。這會發送標準的網路參數（如應用程式版本和語言偏好，以及您的作業系統版本和 CPU 架構）以獲取版本配置清單，但不會收集 any 個人或對話數據。
- **遠端遙測與診斷日誌:** 為了監控服務健康狀況並防止濫用 API 速率限制，更新請求會被記錄到 GCP Cloud Logging 中，該日誌服務會固有地識別請求來源的用戶端 IP 地址。這些日誌會記錄作業系統詳情和匿名安裝 GUID。所有此類日誌都會保留 30 天，然後自動被永久刪除。

## 3. 網站分析與 Cookie 同意
為了分析訪問者流量和監控下載率，本網站使用 Google Analytics (GA4)。預設情況下，分析追蹤完全停用，且不會載入任何 Cookie。

當您訪問本網站時，會顯示一個隱私權同意橫幅：
- **同意接受:** 如果您接受追蹤，我們會動態載入 Google Analytics。Google Analytics 將放置持久性識別碼（Cookie，如 _ga 和 _ga_<container-id>）來追蹤匿名頁面瀏覽和下載點擊。您的 IP 地址會被匿名化。
- **拒絕追蹤:** 如果您拒絕，Google Analytics 腳本將永遠不會載入。不會存儲任何 Cookie，頁不會發送裝置指紋資訊。

您可以隨時撤回或重置您的選擇：[管理隱私偏好](#)（這將清除您儲存的選擇，並立即再次顯示同意橫幅）。

## 4. 共享您的資訊
我們不會出售、出租或交易您的個人數據。我們僅與以下受信任的第三方服務提供商（子處理商）共享運行我們的更新和診斷服務所需的有限技術資訊：
- **雲端基礎設施:** 我們使用 Google Cloud Platform (GCP) Cloud Logging 儲存標準的遙測日誌（例如用於更新檢查的 IP 地址 and 裝置資訊），以用於診斷和安全目的。這些日誌會在 30 天後自動清除。

## 5. 本地資料的安全性
由於您的資料儲存在本地，因此已索引對話的安全性完全取決於您自己裝置的安全性。我們建議您使用標準加密工具（例如 macOS 上的 FileVault）和強密碼來保護您的裝置。
